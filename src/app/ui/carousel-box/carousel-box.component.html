assignment Projects

  1. Neighborhood App

    Published August 2016

    Single page application built using knockout.js and hosted on Github Pages.
    Full page Google map application that display nearby locations listed on Yelp.
    Allows for filtering by distance, categories, subcategories, or you can search by keywords manually.
    Brings up the top results on the map, allowing you to click on them and view ratings, reviews, etc. in the pop up windows on the map

    Click here to try it!

  2. Gwent Card Deck Builder

    Published February 2017

    Browser based application for card selection based on a card game from a popular RPG video game (Witcher)
    Developed using React, Redux, Webpack, Babel and node.
    Allows a user to select a deck type, select individual cards that make up the deck, see statistics on their current deck build, and more.
    Future plan to create a browser port of the actual gwent card game; this is just currently the UI for deck building.

    Click here to try it!

  3. Radioactive Ladybugs

    Published July 2016

    Browser based frogger clone, created with Javascript, HTML5 and CSS3.
    Developed with native Javascript code using the canvas API offered in HTML5 for use across all major browsers.
    Users get 3 lives and score points for making it into the water; if a ladybug hits you, a life is lost.
